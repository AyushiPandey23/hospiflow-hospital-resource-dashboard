# HospiFlow – Hospital Resource Utilization Dashboard

## Problem Statement

Hospitals often face challenges such as high patient load, inefficient bed allocation, lack of real-time monitoring, and fragmented data across systems.  
This project aims to address these issues by transforming raw hospital data into meaningful insights for better decision-making.  

The dashboard provides a centralized view of hospital operations, helping in improving efficiency and optimizing resource utilization.

---

## Steps Followed

- Step 1 : Collected and organized hospital datasets  
- Step 2 : Performed data cleaning and preprocessing using SQL and Excel  
- Step 3 : Loaded cleaned data into Power BI  

- Step 4 : Created data model:
  - Established relationships between multiple tables  
  - Structured data for efficient analysis  

- Step 5 : Designed interactive dashboard using visuals and KPIs  

- Step 6 : Created visuals:
  - Patient trend analysis over time  
  - Department-wise patient distribution  
  - Emergency case analysis by severity  
  - Equipment utilization overview  
  - Diagnosis-based patient insights  

- Step 7 : Implemented KPI metrics:
  - Total Patients  
  - Average Stay Duration  
  - Treatment Cost  
  - Bed Occupancy Rate  

- Step 8 : Applied filters and slicers for dynamic data exploration
